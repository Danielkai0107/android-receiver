新架構：
━━━━━━━━━━━━━━━━━━━━━━━━━
MainActivity (主容器)
  └─ BottomNavigationView (底部導覽列)
      ├─ 📱 執行頁 (HomeFragment)
      ├─ 📋 白名單 (WhitelistFragment)
      ├─ ⚙️ 參數設定 (SettingsFragment) ← 可編輯
      └─ 🔐 權限設定 (PermissionsFragment)

底部導覽列：
━━━━━━━━━━━━━━━━━━━━━━━━━
┌─────────────────────────────┐
│                             │
│   (內容區域 - Fragment)      │
│                             │
├─────────────────────────────┤
│ [執行] [白名單] [參數] [權限] │
└─────────────────────────────┘

1️⃣ 執行頁 (HomeFragment)
━━━━━━━━━━━━━━━━━━━━━━━━━
- Gateway ID 顯示
- 統計資訊（白名單、掃描、上傳、待上傳）
- [▶️ 開始掃描] / [⏹ 停止掃描]
- [📋 查看掃描清單]
- [🔄 手動同步白名單]（自動同步仍運作）

2️⃣ 白名單頁 (WhitelistFragment)
━━━━━━━━━━━━━━━━━━━━━━━━━
- 顯示所有白名單設備
- 設備數量統計
- UUID、Major、Minor、MAC
- 同步時間

3️⃣ 參數設定頁 (SettingsFragment) - 可編輯！
━━━━━━━━━━━━━━━━━━━━━━━━━
- 掃描頻率（3/5/7/10 秒）
- 上傳間隔（30/60/90/120 秒）
- 白名單同步間隔（5/10/15/30 分鐘）
- GPS 更新頻率（1/2/3/5 分鐘）
- 離線快取上限（500/1000/1500/2000 筆）

點擊任一項目會彈出選擇對話框！✅

4️⃣ 權限設定頁 (PermissionsFragment)
━━━━━━━━━━━━━━━━━━━━━━━━━
- 📍 位置權限 + [授予] 按鈕
- 📡 藍牙權限 + [授予] 按鈕
- 📱 讀取手機狀態 + [授予] 按鈕
- 🔔 通知權限 + [授予] 按鈕
- 🔙 背景位置權限 + [授予] 按鈕
- [⚙️ 打開系統設定]

每個權限都有獨立的授予按鈕！✅

特色功能：
━━━━━━━━━━━━━━━━━━━━━━━━━

測試步驟：
━━━━━━━━━━━━━━━━━━━━━━━━━
1. 在 Android Studio 中 Run (Ctrl+R)

2. 在手機上瀏覽底部 4 個 Tab：
   
   📱 執行頁：
   - 查看統計
   - 點「開始掃描」
   - 點「查看掃描清單」
   
   📋 白名單頁：
   - 查看白名單設備
   
   ⚙️ 參數設定頁：
   - 點任一參數項目
   - 從對話框選擇新值
   - 立即生效！
   
   🔐 權限設定頁：
   - 查看所有權限狀態
   - 點「授予」按鈕請求權限
   - 或點「打開系統設定」

🎉 完整的 Android 接收器 App！
