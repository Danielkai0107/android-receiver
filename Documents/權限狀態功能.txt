權限說明：
━━━━━━━━━━━━━━━━━━━━━━━━━
📍 位置權限（必要）
   - 用途：藍牙掃描必須、GPS 定位
   - 未授予：無法掃描 Beacon
   
📡 藍牙權限（必要）
   - 用途：掃描 BLE Beacon
   - 未授予：無法掃描 Beacon
   
📱 讀取手機狀態（必要）
   - 用途：獲取 IMEI 作為 Gateway ID
   - 未授予：使用 Android ID 作為備用
   
🔔 通知權限（必要，Android 13+）
   - 用途：前景服務通知
   - 未授予：服務可能被系統終止
   
🔙 背景位置權限（建議）
   - 用途：App 在背景時繼續定位
   - 未授予：前景時仍可正常使用

功能特色：
━━━━━━━━━━━━━━━━━━━━━━━━━
✅ 即時更新：每次回到主畫面都會更新
✅ 顏色標示：
   - 綠色 ✅ = 已授予
   - 紅色 ❌ = 未授予（必要）
   - 橙色 ⚠️ = 未授予（建議）
✅ 整體狀態：一眼看出是否所有必要權限都已授予
✅ 快速診斷：方便排查問題

測試步驟：
━━━━━━━━━━━━━━━━━━━━━━━━━
1. 在 Android Studio 中 Run (Ctrl+R)
2. 查看主畫面的權限狀態區域
3. 如有未授予的權限，點擊「開始掃描」會自動請求
4. 授予權限後，返回主畫面查看狀態更新

🎉 現在更容易診斷問題了！
